Projects

YaaS

Youtube as a Service (YaaS) is a Django web app that converts files into YouTube videos for virtually unlimited, lossless storage. I designed a compression-tolerant pipeline with 99.9% bit-level recovery accuracy. With the current design, each frame in the video can store up to 129,600 bytes!

Resume Builder

Resume Builder is a Python-based tool that uses OpenAI to analyze job descriptions, extract key terms, and rewrite the inputted resume content. It then generates a polished LaTeX resume tailored to the target job. This can improve ATS keyword match rates significantly, increasing the chances of receiving more interview opportunities.

Organiser

Organizer App is a productivity tool built in Unity, designed to help me as a students manage their time, tasks, and academic goals more effectively.
